# Household Chore RPG (Browser App)

A simple browser-based app to gamify household chores as quests, earn experience points, and level up.

## How to use
1. Open `index.html` in your browser.
2. Add chores as quests.
3. Mark quests complete to earn XP.
4. Watch the level and progress update.

## Files
- `index.html`: Main page structure.
- `style.css`: App styling.
- `script.js`: Quest/task logic and XP leveling.
